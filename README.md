*A k8s Auto Web deploy project*
---
When you change index.html on your device and commit it, GitHub Actions builds a new image and updates Kubernetes. If a pod crashes, Kubernetes restarts it automatically.

Managed the entire development lifecycle from a Linux-based mobile environment (Termux), using Git CLI for version control and GitHub Actions for the remote CI/CD pipeline.
